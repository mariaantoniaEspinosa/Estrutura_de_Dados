## VS - Code 
### Instalar as extensões que forem solicitadas quando começar um código em determinada linguagem. 
### Caminho para compilar e executar no TERMINAL:

- cd Downloads\Estrutura_De_Dados (para localizar)
  
- dir (para saber a pasta que se encontra o arquivo salvo)

- gcc Nome.c -o Nome (para compilar códigos em C)

- g++ Nome.cpp -o Nome (para compilar códigos em C++)

- .\Nome (para executar)
